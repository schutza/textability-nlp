# Dialog Acts

A library for the detection of dialogue acts.

Prerequisites:

- Download language resources
    ```
    spacy download en_core_web_sm
    spacy download de_core_news_sm
    ```
  
## Testing

Run unit tests using `pytest` by executing

    pytest
